### 目的

> 学习 `Spring` 的深层原理，参考资料:Spring源码深度解读

### 如何运行

#### 环境

*   安装 `maven` 
*   安装 `jdk1.8` 
*   安装 `lombok` 插件，安装步骤 [Intellij IDEA 安装lombok及使用详解](https://blog.csdn.net/zhglance/article/details/54931430)

#### 运行

1、 添加sql到数据库 `mysql 5.7.22` 中
2、 运行 `top.github.web.App` 即可

#### 特性

*   [x] Aop + ioc
*   [x] 申明式事务
*   [x] 接入mybatis
*   [x] 支持url注解
*   [x] json 响应
*   [x] 跨域
*   [x] 文件上传
*   [ ] 事务传播行为
